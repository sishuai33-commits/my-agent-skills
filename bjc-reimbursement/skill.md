# BJC差旅报销 Skill

## 功能概述

为佰钧成员工提供差旅报销全流程辅助：
- 输入 BTA 备案信息 → 自动生成标准化文件夹和材料
- 提供每步骤操作指引 → 避免每次重新理解流程
- 生成邮件模板 → 快速完成线下环节

---

## 使用方式

### 触发条件
用户提到以下关键词时自动激活：
- "报销"、"差旅报销"、"BJC报销"、"佰钧成报销"、"出差报销"

### 输入信息

用户需要提供：

| 信息项 | 示例 | 来源 |
|--------|------|------|
| BTA备案编号 | BTA-20250225-001 | BTA系统 |
| 出差日期 | 2025.02.25-2025.02.28 | BTA系统 |
| 出差地点 | 深圳-南京 | BTA系统 |
| 出差事由 | 项目现场支持 | BTA系统 |
| 费用构成项 | 飞机票1420、住宿654、补助746 | 发票+计算 |
| 发票明细 | 类型、金额、票号、开票日期 | 实际发票 |

## OA操作步骤指引

### 步骤1：选择报销类型
- 路径：财务会计 → 财务报销 → 客户付费差旅费报销申请
- 选择正确的报销类型

### 步骤2：填写基本信息
| 项目 | 内容来源 |
|------|----------|
| 报销主题 | [出差日期]-[地点]出差报销 |
| BTA备案编号 | 用户输入 |
| 出差日期 | 用户输入 |
| 出差天数 | 自动计算 |

### 步骤3：选择费用类别
可选类别（根据实际费用勾选）：
- 长途交通费
- 住宿费
- 出差生活补助

**注意**：备注必须填写

### 步骤4：填写发票信息
| 费用项 | 科目路径 | 发票类型选择 |
|--------|----------|--------------|
| 长途交通费（飞机） | 主营业务成本-客户付费差旅费-国内差旅费-长途交通费 | 其它发票 |
| 长途交通费（高铁） | 同上 | 其它发票 |
| 住宿费 | 主营业务成本-客户付费差旅费-国内差旅费-住宿费 | 全电发票普票 |
| 出差生活补助 | 主营业务成本-客户付费差旅费-国内差旅费-出差生活补助 | 其它发票 |

费用编号格式：BZ-{年月日}-{客户编号}-{序号}（OA自动生成，需回填记录）

### 步骤5：上传附件
必须上传：
1. 客户付费差旅费报销单 Excel
2. 发票扫描件/PDF

### 步骤6：分摊/归集
- 子项目编号：固定（D-202512-188896）或用户指定
- 分摊部门：固定（营业部-BU6-GTS企业业务组）或用户指定

### 步骤7：受款信息
- 受款人：员工本人（自动匹配）
- 个税基数调整金额计算公式：
  ```
  个税基数调整金额 = 出差生活补助金额 - 出差天数 × 100
  ```

### 步骤8：提交审批
检查无误后提交

## 协作流程

### 初次使用
1. 用户触发 Skill，提供输入信息
2. Skill 生成文件夹和材料包
3. 用户按指引操作 OA，截图存档
4. 用户回填费用编号等动态信息
5. Skill 更新记录文档

### 后续迭代
- 用户提出调整 → Skill 更新流程/模板
- 发现新规律 → 补充到 Skill 知识库
- 其他员工使用 → 扩展适用范围

---

## 注意事项

1. **费用编号**：OA 系统自动生成，不是人工编制
2. **发票类型**：飞机票/补助选"其它发票"，住宿选"全电发票普票"
3. **备注必填**：每项费用都要写备注
4. **个税计算**：有固定公式，自动计算
5. **分摊信息**：同一出差通常归集到同一项目

## 执行规则

当用户触发此 Skill 时：

1. **询问必要信息**：BTA备案编号、出差日期、费用明细、发票信息
2. **创建文件夹**：按标准结构创建
3. **生成报销流程记录.md**：预填充固定信息，标注待填项
4. **生成邮件模板**：预留填空位置
5. **提供操作指引**：按步骤顺序说明
6. **等待用户反馈**：回填动态信息、提出调整

后续使用中根据用户反馈持续优化 Skill 内容。
